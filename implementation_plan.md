# Merleau-Ponty Dashboard - Critique & Future Plan

본 문서는 현재까지 개발된 메를로-퐁티 연구 대시보드의 현재 상태를 점검하고, 발견된 오류의 원인 및 해결책, 그리고 향후 대시보드의 완성도를 높이기 위한 **추가/제거 기능 리스트**를 정리한 계획서입니다. GitHub 리포지토리에 업로드하여 프로젝트 로드맵으로 활용할 수 있습니다.

## 1. 현재 대시보드 비판 및 수정 사항 (Critiques & Fixes)

### 🚨 [Critical] PyVis 네트워크 그래프 `[WinError 5] Access Denied: 'lib'` 오류
* **원인**: `pyvis` 라이브러리는 네트워크 그래프 HTML을 렌더링할 때, 자바스크립트와 CSS 자산(vis.js 등)을 담을 `lib` 폴더를 실행 환경(현재 디렉토리)에 자동으로 생성하려고 시도합니다. 그러나 현재 폴더의 권한 문제(Windows UAC 등)로 인해 폴더 생성 접근이 거부(Access Denied)되어 프로그램이 중단되었습니다.
* **해결책 (Fix)**: `Network()` 객체를 생성할 때, 로컬 폴더를 만들지 않고 외부 웹(CDN)에서 자산을 불러오도록 `cdn_resources='remote'` 옵션을 부여합니다. 이를 통해 권한 충돌 없이 그래프를 렌더링할 수 있습니다.

### ⚠️ UX 및 데이터 시각화의 한계
* **비판**: 현재 데이터프레임(Raw Data Explorer)에 너무 많은 컬럼이 노출되어 있어 가독성이 떨어집니다.
* **비판**: 슬라이더로 연도만 조절할 수 있어, 특정 학술지나 저자만 좁혀서(Drill-down) 보고 싶을 때 불편합니다.

## 2. 기능적으로 추가할 사항 (Features to Add)

향후 대시보드의 '연구 지원 도구'로서의 역할을 강화하기 위해 다음 기능을 추가합니다.

* **[NEW] 다중 필터링 시스템 (Multi-Select Filters)**
  * 사이드바에 '저자명' 및 '주제분야'를 선택할 수 있는 멀티 셀렉트 박스를 추가하여, 특정 연구자의 문헌만 필터링하는 기능을 도입합니다.
* **[NEW] 키워드 트렌드 딥다이브 (Keyword Deep-dive)**
  * 특정 키워드(예: '살', '지각')를 선택하면, 해당 키워드의 연도별 인용수 추이만 단독으로 보여주는 상세 분석 기능을 추가합니다.
* **[NEW] AI 기반 텍스트 분석 고도화 (LLM Integration)**
  * 현재 버튼만 있는 'AI 초록 요약' 기능에 실제 텍스트 요약 알고리즘(또는 외부 API 연동)을 붙여, 논문을 읽지 않고도 핵심 주장을 파악할 수 있게 합니다.

---

## 3. 기능적으로 뺄 사항 (Features to Remove)

대시보드의 목적성을 명확히 하고 성능을 최적화하기 위해 불필요한 요소를 제거합니다.

* **[DELETE] 불필요한 원시 데이터 컬럼 출력**
  * Raw Data 테이블에서 'KCI 등재 구분', '반출일', 'URL' 등 분석에 직관적인 도움을 주지 않는 메타데이터 컬럼은 화면에서 숨김(Drop) 처리하여 깔끔한 UI를 유지합니다.
* **[DELETE] 과도한 물리 시뮬레이션 (Network Graph)**
  * 만약 `cdn_resources='remote'` 수정 후에도 네트워크 그래프가 앱 전체 로딩 속도를 심각하게 저하시키거나 UI를 해친다면, 해당 탭을 과감히 **제거**하고 대신 정적인 매트릭스 표나 히트맵(Heatmap)으로 키워드 연관성을 대체합니다. (우선순위: 수정 후 경과 관찰 -> 문제시 제거)
