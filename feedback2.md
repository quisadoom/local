# 🚀 Dashboard Technical Roadmap: Merleau-Ponty Research
> **현상학적 데이터를 정보의 '살(Chair)'로 전환하기 위한 기술적 고도화 및 기능 확장 제안**

---

## 1. ⚙️ 데이터 파이프라인 및 성능 최적화 (3A: Automation)
현재 870여 건의 데이터를 처리하는 속도를 개선하고, 데이터의 정밀도를 높이기 위한 기술적 조언입니다.

* **한국어 형태소 분석기 도입**: 현재의 정규식(Regex) 기반 키워드 추출은 '신체와', '신체는' 같은 조사를 완벽히 분리하기 어렵습니다. `Kiwi`나 `KoNLPy` 같은 형태소 분석기를 적용하여 **명사 단위의 정밀한 키워드 추출**을 권장합니다.
* **SQLite 인덱싱(Indexing)**: 논문 제목, 저자, 초록 컬럼에 인덱스를 생성하면 검색 및 필터링 속도가 수 밀리초(ms) 단위로 빨라집니다.
* **데이터 캐싱 전략**: Streamlit의 `@st.cache_data`를 활용해 DB 쿼리 결과를 메모리에 상주시켜, 사용자가 필터를 바꿀 때마다 발생하는 리로딩 지연을 최소화하세요.

---

## 2. 🎨 시각화 및 UX 고도화 (Concept & Evolution)
단순한 차트를 넘어 철학적 맥락을 시각적으로 구현하는 방법입니다.

* **인터랙티브 네트워크 그래프 (Pyvis/NetworkX)**: 
    * 사례로 본 'Existentialism & Freedom' 대시보드처럼 키워드 간의 연결망을 구축하세요.
    * **살(Chiasme)** 개념을 시각화하여 특정 키워드(예: 신체)가 다른 학문(예: 인공지능)과 어떻게 얽히는지 보여주는 노드-링크 다이어그램을 추천합니다.
* **시대별 담론 스트림그래프 (Streamgraph)**: 
    * 연도별로 '지각', '정치', '존재론' 등 주제어의 점유율 변화를 물결 모양의 그래프로 시현하여 메를로-퐁티 연구의 패러다임 변화를 시각화합니다.
* **다크 퍼플(#3F0099) 디자인 시스템**: 
    * 차트의 포인트 컬러와 폰트 가독성을 통일하여 '철학적 무게감'을 유지하는 일관된 디자인 가이드를 적용하세요.

---

## 3. 📝 연구 및 집필 지원 기능 (4D: Discernment & Agency)
졸업 논문 작성이라는 최종 목적지에 실질적인 도움을 주는 기능입니다.

* **연구 공백 탐지기 (Research Gap Finder)**: 
    * 데이터상 인용 수는 높지만 최근 연구가 뜸한 주제나, 결합 빈도가 현저히 낮은 두 키워드의 조합(예: 메를로-퐁티 + 양자역학)을 자동으로 추천해주는 기능을 추가하세요.
* **서지 정보 내보내기 (Export to BibTeX/CSV)**: 
    * 선택한 논문 리스트를 BibTeX나 EndNote 형식으로 바로 내려받아 논문 작성 시 참고문헌 정리에 드는 시간을 단축하세요.
* **초록 요약 및 키워드 매칭**: 
    * 특정 논문을 클릭하면 AI가 해당 초록을 3줄로 요약해주고, 현재 사용자의 논문 주제와 얼마나 밀접한지(Similarity Score) 보여주는 기능을 고려해보세요.

---

## 4. 🌐 확장성 및 배포 전략
* **Netlify/Streamlit Cloud 배포**: 로컬(`localhost`)을 넘어 외부에서도 접근 가능한 URL을 확보하여 지도 교수님이나 동료들과 연구 성과를 공유하세요.
* **API 연동**: KCI 오픈 API를 직접 연동하여, 매번 Excel을 업로드할 필요 없이 버튼 클릭 한 번으로 최신 데이터를 동기화하는 기능을 구현할 수 있습니다.
